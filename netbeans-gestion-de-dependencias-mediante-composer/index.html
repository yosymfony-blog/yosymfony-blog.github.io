Netbeans: Gestión de dependencias mediante Composer

Posted by victor on March 13th, 2013
Hace unas semanas que se lanzó la versión 7.3 y el desarrollo continúa con nuevas características para la siguiente versión. En este caso, el equipo de desarrollo del módulo PHP de Netbeans ha añadido el soporte para la gestión de dependencias mediante Composer. Por si no lo conoces aún, Composer es un gestor de dependencias, una aplicación en la que indicamos que librerías o componentes son necesarios para nuestro proyecto y se encarga de obtenerlas y mantenerlas actualizadas. Esta nueva característica se encuentra en una implementación muy temprana y es susceptible de mejoras, sobre todo en rendimento. El equipo de desarrollo nos invita a descargar la versión de desarrollo y comunicar los bug encontrados. El funcionamiento es sencillo: En el navegador de proyectos seleccionamos el proyecto y, al obtener las opciones con el botón derecho, encontraremos una llamada "Composer". Desde ella podremos añadir, actualizar, validar etc...
